Conception d'un Cockpit de Simulateur de Vol

Projet S3 & S4 - IUT de Cachan | Responsable : Denis PENARD

KiCad (CAO) STM32 IDE Python X-Plane 11 HID Interface

Introduction

Ce projet majeur de deuxième année visait à concevoir et réaliser une interface matérielle fonctionnelle pour le simulateur de vol de l'IUT de Cachan. L'objectif était de créer un panel de contrôle réaliste, capable de faire le lien entre le matériel physique et le logiciel de simulation professionnel X-Plane 11, notamment pour le pilotage d'un hélicoptère.

Test final en immersion dans le simulateur de vol de l'IUT.

Déroulement du projet

Le projet s'est déroulé sur l'ensemble des semestres 3 et 4. Le semestre 3 a servi d'introduction avec la réalisation d'un PCB et d'une interface homme-machine via STM32 IDE, permettant d'envoyer des données depuis un appareil vers l'ordinateur. Nous avons configuré le système comme un clavier d'ordinateur pour envoyer les informations des boutons vers le simulateur X-Plane afin de contrôler l'hélicoptère.

Au semestre 4, le travail en binôme consistait à compléter le Glass Cockpit de l'appareil. Sur la base existante, il manquait le tachymètre, l'écran et les voyants d'alerte. Notre groupe s'est spécifiquement chargé de la conception et de l'intégration du bloc des voyants d'alerte.

1. Réalisation du PCB

1.1 Conception sous KiCad

La phase de CAO a permis de structurer l'architecture électronique du panel. J'ai conçu le schéma électrique en intégrant la gestion des signaux pour les commutateurs et le rétroéclairage. Le routage a été optimisé pour une fabrication double-face.

1.2 Brasage des composants

L'étape de brasage a consisté à assembler les composants critiques sur la carte. J'ai intégré des LEDs de couleurs différentes en respectant la datasheet du véritable hélicoptère Cabri G2. L'électronique repose sur un microcontrôleur STM32L073, alimenté via un port USB-C avec un régulateur de tension 3,3V.

2. Programmation

2.1 Logiciel STM32 IDE

La programmation embarquée gère l'acquisition des entrées numériques et pilote les sorties pour les voyants d'alerte via une pile USB HID native.

2.2 Code Python de transmission

Un script Python assure le lien en temps réel entre X-Plane 11 et notre panel, permettant d'allumer les voyants d'alerte en fonction des incidents simulés.
